# `co2-sensor`

A CO2, temperature, humidity and air pressure sensor.

## Dependencies

#### 1. `flip-link`:

```console
$ cargo install flip-link
```

#### 2. `probe-rs`:

``` console
$ # make sure to install v0.2.0 or later
$ cargo install probe-rs --features cli
```

## Run

```
cargo run --release
```
